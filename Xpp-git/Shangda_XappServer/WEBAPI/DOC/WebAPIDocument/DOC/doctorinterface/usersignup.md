##用户手机注册

|相关信息|内容|
|--|--|
|访问类型|POST|
|访问地址|/api/doctor/SignUp|

###请求参数

|参数名称|是否必填|类型|说明|示例数据|
|--|--|--|--|--|--|
|phone|是|字符串|移动电话||
|Password|是|字符串|密码||
|ValidCode|是|字符串|验证码||


###请求示例
<pre>
<code>

"Password":"123456789",
"ValidCode":"1154"
}
</code>
</pre>
